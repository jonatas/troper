h2. Introdução

* Aplicativos na internet
** Todos os dias vemos um novo aplicativo na internet
** A Leosoft tem muitos desses

* A partir de um feedback

Projetos de software geralmente apresentam foco em algumas partes e vantagens para as mesmas. A Leosoft sempre teve um bom feedback em um de seus sistemas por possuir uma ferramenta para construir relatórios de maneira fácil e útil. Os usuários se sentem bem por ter liberdade para construir/modificar relatórios para suas necessidades específicas.

h2. Objetivos (geral)

Abraçar todas as idéias na construção de um software não dá certo, o objetivo do projeto é encontrar um escopo para a construção de relatórios simples e personalizados.


h3. Construir relatórios  (objetivos específicos)

Uma inteface estilo 'passo a passo' para construir um relatório.

h3. Executar relatórios (objetivos específicos)

Uma interface gerada automaticamente apartir dos filtros pré-definidos para o relatório, pegando os dados do mesmo, enviando para o servidor processar, executar os filtros e construir a interface com os dados no formato html.

h2. Justificativa

* Necessidade da ferramenta na web 

A maioria dos projetos web da Leosoft são construídos usando o framework Ruby On Rails, e este não possui nenhuma ferramenta pronta para k sendo este de fácil manutenção e flexibilidade. A adição módulos plugáveis pode ser fácilmente introduzida nestes em qualquer sistema.
 
* Personalização - Cada pessoa gosta de uma cor, ver o nome antes do telefone, não usa o telefone e apenas celular. Dificilmente um sistema irá agradar a todos os usuários. Essa interface irá trazer ao usuário uma forma para fazer estas mudanças sem a necessidade de um pedido aos desenvolvedores do software.

h2. Valores agregados ao sistema

**Ser plugável**

Irá funcionar para os sistemas existentes da Leosoft. Isto agregará valor a outros sistemas que já possuem valor para os clientes.

** Ser flexível**

Entender os modelos e regras do sistema. Possibilidade de acoplar o dicionário de dados ou a fonte de dados.

h3. Leosoft

A empresa terá uma diminuição significativa na demanda de construção e manutenção de relatórios. Poderá oferecer mais um módulo interessante para o cliente.
